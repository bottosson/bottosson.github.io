Interactive color picker comparison

Almost all color pickers are based on HSL or HSV, but that might not be the best choice. Here standard HSV and HSL color pickers are compared with Oklch, Okhsv, Okhsl and Hsluv.

For background and more information about the color spaces used, check out the accompanying blog post.

The source code is available on Github here.
